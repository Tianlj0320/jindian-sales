## V1.2.0 — 2026-04-19

### 🐛 Bug 修复
| # | 问题 | 根因 | 解决方案 |
|---|------|------|---------|
| 1 | 登录页白屏 | `JSON.parse(LS('currentUser'))` 失败时无 try-catch，`localStorage` 存的是对象而非字符串 | 改用三目运算符保护 `JSON.parse` |
| 2 | Vue Codegen 错误（白屏） | 员工资料页表格后有孤儿元素（复制残留） | 删除 `</el-table>` 后的游离代码 |
| 3 | `/` 路径返回 API JSON | `root()` 直接返回 JSON dict | 改为 `RedirectResponse(url="/static/index.html")` |
| 4 | 登录弹窗不显示 | 弹窗 HTML 放在 `#app` div 外面 | 移入 `#app` 内 |

### 🔧 技术改进
- **el-table → 原生 HTML 表格**：Element Plus v2.4.0 + Vue 3.4.0 的 colspan 计算有 bug，4个基础资料页全部换原生表格
- **Vue.js 换生产版本**：运行时不走开发编译器检查，Codegen 警告消除
- **.gitignore 完善**：增加 `app/*/__pycache__/` 忽略规则

### ✨ 新增功能
- **打印功能**（`app/api/print_api.py`）：4种模板
  - `contract` — 订货合同
  - `measurement` — 测量记录单
  - `processing` — 加工单
  - `install` — 安装派工单
- **基础资料4页筛选器**：
  - 产品资料：供应商筛选 + 布版筛选
  - 供应商管理：类型筛选（成品/布艺/辅料）
  - 员工资料：职务筛选
  - 客户资料：客户类型筛选（零售/工程/分销）
- **基础资料4页搜索框**：全部加搜索（名称/编号/联系人等字段）
