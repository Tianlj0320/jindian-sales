# 窗帘销售系统 V3.0 · API 接口文档

> 版本：V3.0 | 更新：2026-04-21

## 一、订单模块

### 1.1 创建订单

**路径：** `/api/v3/orders`  **方法：** `POST`

**请求参数：**

| 参数名 | 类型 | 必须 | 说明 |
|--------|------|------|------|
| customer_id | integer | 是 | 客户ID |
| order_type | string | 是 | 订单类型（窗帘/墙布/全屋） |
| items | array | 是 | 订单明细数组 |
| items[].product_id | integer | 是 | 产品ID |
| items[].window_location | string | 否 | 窗户位置 |
| items[].supplier_id | integer | 否 | 供应商ID |
| items[].material_type | string | 否 | 物料类型（主料/辅料） |
| items[].calculation_method | string | 否 | 计算方式 |
| items[].width | number | 否 | 宽度（米） |
| items[].height | number | 否 | 高度（米） |
| items[].pleat_ratio | number | 否 | 褶皱倍数 |
| items[].qty | number | 是 | 数量 |
| items[].unit_price | number | 是 | 单价 |
| salesperson | string | 否 | 导购 |
| delivery_date | string | 否 | 交货日期 |
| delivery_method | string | 否 | 提货方式 |
| install_address | string | 否 | 安装地址 |

**响应：**

| 状态码 | 说明 |
|--------|------|
| 201 | 创建成功 |
| 400 | 参数错误 |
| 500 | 服务器错误 |

**返回值结构：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1001,
    "order_no": "2604210001",
    "status_key": "created",
    "created_at": "2026-04-21 13:00:00"
  }
}
```

**示例请求：**
```json
{
  "customer_id": 5,
  "order_type": "窗帘",
  "items": [
    {
      "product_id": 101,
      "window_location": "客厅",
      "supplier_id": 3,
      "material_type": "主料",
      "calculation_method": "按米",
      "width": 3.0,
      "height": 2.8,
      "pleat_ratio": 1.5,
      "qty": 1,
      "unit_price": 120.00
    }
  ],
  "salesperson": "张三",
  "delivery_date": "2026-05-10"
}
```

### 1.2 确认订单（触发采购拆分）

**路径：** `/api/v3/orders/{order_no}/confirm`  **方法：** `POST`

**说明：** 订单确认后自动触发采购拆分流程

**请求参数：** 无

**响应：**

| 状态码 | 说明 |
|--------|------|
| 200 | 确认成功 |
| 404 | 订单不存在 |
| 409 | 状态不允许确认（必须是created） |
| 500 | 服务器错误 |

**返回值结构：**
```json
{
  "code": 0,
  "message": "订单已确认并进入采购拆分",
  "data": {
    "order_no": "2604210001",
    "status_key": "confirmed",
    "purchase_orders": [
      { "po_no": "PO260421001", "supplier_name": "浙江面料厂", "material_type": "主料", "amount": 1500.00 },
      { "po_no": "PO260421002", "supplier_name": "轨道配件商", "material_type": "辅料", "amount": 300.00 }
    ]
  }
}
```

## 三、生产反馈模块

| 500 | 服务器错误 |

## 七、通用响应格式

所有API统一返回以下格式：

```json
{
  "code": 0,
  "message": "success",
  "data": { ... }
}
```

| code | 说明 |
|------|------|
| 0 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 404 | 资源不存在 |
| 409 | 状态冲突（如状态不允许此操作） |
| 500 | 服务器内部错误 |

---

## 八、错误码说明

| code | message | 说明 |
|------|---------|------|
| 1001 | ORDER_NOT_FOUND | 订单不存在 |
| 1002 | ORDER_STATUS_CONFLICT | 订单状态不允许此操作 |
| 2001 | PO_NOT_FOUND | 采购单不存在 |
| 2002 | PO_ALREADY_PROCESSED | 采购单已处理 |
| 3001 | FEEDBACK_NOT_FOUND | 反馈单不存在 |
| 4001 | INS_NOT_FOUND | 安装单不存在 |
| 4002 | INS_ALREADY_ASSIGNED | 安装单已分配 |
| 5001 | CUSTOMER_NOT_FOUND | 客户不存在 |
