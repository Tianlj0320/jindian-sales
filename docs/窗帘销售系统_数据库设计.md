# 窗帘销售系统 · 数据库设计

> 基于 SQLAlchemy ORM 模型自动生成，数据类型兼容 SQLite
> 对应模型文件：`app/models.py`
> 版本：V3.0 | 更新：2026-04-21

---

## ER 关系图（V3.0 完整版）

```
customers（客户）
  ├── orders（订单）
  │     ├── order_items（订单明细）← 改造：新增 supplier_id, material_type
  │     ├── purchase_orders（采购单）← 新增
  │     │     └── purchase_order_items（采购单明细）← 新增
  │     ├── production_feedback（生产反馈）← 新增
  │     ├── measure_records（测量记录）← 新增
  │     └── installation_orders（安装单）← 新增
  │           └── employees（员工/安装工）
  └── follow_records（跟进记录）

products（产品）── suppliers（供应商）
  └── fabric_categories（布版分类）

installer_account（安装工账户）── installation_orders（安装单）
```

## 四、SQL 建表脚本（V3.0）

```sql
-- ============================================================
-- 窗帘销售系统 V3.0 数据库建表脚本
-- 对应 app/models.py（SQLAlchemy）
-- ============================================================

-- V3.0 状态key枚举：
-- created, confirmed, split, purchasing, arrived,
-- producing, production_exception, production_done,
-- installation_generated, shipped, installed, accepted, closed

-- ---------- 改造表：orders ----------
CREATE TABLE orders (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    order_no VARCHAR(20) UNIQUE NOT NULL,
    customer_id INTEGER REFERENCES customers(id),
    customer_name VARCHAR(50),
    customer_phone VARCHAR(20),
    order_type VARCHAR(20),
    status VARCHAR(20) DEFAULT '待确认',
    status_color VARCHAR(20) DEFAULT 'pending',
    status_key VARCHAR(30) DEFAULT 'created',
    quote_amount DECIMAL(12,2) DEFAULT 0,
    discount_amount DECIMAL(12,2) DEFAULT 0,
    round_amount DECIMAL(12,2) DEFAULT 0,
    amount DECIMAL(12,2) DEFAULT 0,
    received DECIMAL(12,2) DEFAULT 0,
    debt DECIMAL(12,2) DEFAULT 0,
    order_date VARCHAR(20),
    delivery_date VARCHAR(20),
    delivery_method VARCHAR(20),
    salesperson VARCHAR(50),
    history JSON DEFAULT '[]',
    items JSON DEFAULT '[]',
```
